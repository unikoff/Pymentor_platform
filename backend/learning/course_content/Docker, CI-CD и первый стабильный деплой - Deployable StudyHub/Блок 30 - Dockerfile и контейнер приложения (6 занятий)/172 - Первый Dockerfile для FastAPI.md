## 172. Первый Dockerfile для FastAPI

### Цель

Собрать минимальный image StudyHub и запустить API внутри container.

### Почему тема появляется сейчас

После модели Docker нужен самый короткий рабочий путь без оптимизаций.

### Основное содержание

- `FROM`;
- `WORKDIR`;
- `COPY`;
- `RUN`;
- `CMD`;
- установка dependencies;
- build image;
- run container;
- привязка host port;

### Практика StudyHub

Создать первый Dockerfile, собрать image, запустить StudyHub и открыть Swagger с host-машины.

### Визуальная и интерактивная подача

Dockerfile как маршрут сборки; CodeSequence правильного порядка инструкций; TerminalDemo build/run.

### Результат занятия

Первый собственный image StudyHub отвечает на HTTP request.

### Граница занятия

Пока не оптимизировать layers и размер. Сначала нужен прозрачный рабочий вариант.
