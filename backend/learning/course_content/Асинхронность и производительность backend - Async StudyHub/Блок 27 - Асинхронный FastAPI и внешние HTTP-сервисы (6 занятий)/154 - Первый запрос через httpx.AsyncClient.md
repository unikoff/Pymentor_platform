## 154. Первый запрос через httpx.AsyncClient

### Цель

Сделать неблокирующий HTTP request к внешнему сервису.

### Основное содержание

- `httpx.AsyncClient`;
- `async with`;
- `await client.get`;
- status;
- JSON parsing;
- base URL;
- query parameters;
- закрытие клиента;

### Практика StudyHub

Подключить локальный mock-service рекомендаций и получить данные по course id.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

StudyHub получает внешний JSON без блокировки loop.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
