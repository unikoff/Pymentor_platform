## 143. asyncio.run, event loop и первый await

### Цель

Запустить coroutine и увидеть передачу управления во время ожидания.

### Основное содержание

- `asyncio.run`;
- entry coroutine `main`;
- event loop;
- `await`;
- `asyncio.sleep`;
- suspend/resume;
- путь вызова;
- await только внутри async def;

### Практика StudyHub

Написать `main`, вызвать одну async-операцию и подписать pause/resume.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

Первая корректно запущенная async-программа.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
