## 6. SQLite и реальное выполнение migration

Поддержка ALTER TABLE зависит от версии SQLite. Alembic может использовать batch mode, поэтому migration нужно запускать на той же среде, а не только читать.

### Точная модель

- **Контекст:** StudyHub уже находится в состоянии, описанном в паспорте занятия.
- **Механизм:** раздел «SQLite и реальное выполнение migration» добавляет одну новую техническую идею.
- **Контракт:** до выполнения называются вход, ожидаемый результат и допустимые ошибки.
- **Граница:** механизм не подменяет ответственность соседнего слоя.

### Пример 1

```text
def upgrade() -> None:
    with op.batch_alter_table(
        "tasks"
    ) as batch_op:
        batch_op.add_column(
            sa.Column(
                "description",
                sa.String(1000),
                nullable=True,
            )
        )


def downgrade() -> None:
    with op.batch_alter_table(
        "tasks"
    ) as batch_op:
        batch_op.drop_column(
            "description"
        )
```
