## 135. Почему запрос становится медленным

### Цель

Сначала измерять запрос, а не оптимизировать по ощущению.

### Основное содержание

- размер таблицы;
- селективность;
- Seq Scan;
- execution time;
- SQL logging;
- повторяемый benchmark;
- baseline;
- маленькая таблица скрывает проблему;

### Практика StudyHub

Сгенерировать данные, измерить поиск по email/title и зафиксировать baseline.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

Есть воспроизводимый запрос и таблица измерений.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
