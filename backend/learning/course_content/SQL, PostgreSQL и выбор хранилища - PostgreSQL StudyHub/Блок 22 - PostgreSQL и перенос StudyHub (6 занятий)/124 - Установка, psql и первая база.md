## 124. Установка, psql и первая база

### Цель

Запустить PostgreSQL и выполнить первые команды клиентом.

### Основное содержание

- проверка службы;
- `psql`;
- подключение по host/port;
- `\l`, `\c`, `\dt`, `\d`;
- создание базы;
- создание таблицы;
- GUI как дополнительный клиент;
- connection refused;

### Практика StudyHub

Создать `studyhub_dev`, подключиться, создать тестовую таблицу и найти её через psql и GUI.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

Ученик самостоятельно подключается к локальному серверу.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
