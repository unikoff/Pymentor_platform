## 127. Alembic на чистой PostgreSQL-базе

### Цель

Восстановить схему проекта только из истории миграций.

### Основное содержание

- чистая база;
- `alembic current`;
- `alembic history`;
- `upgrade head`;
- metadata;
- revision;
- `alembic_version`;
- ошибка миграции;

### Практика StudyHub

Удалить тестовую базу, создать заново, применить миграции и проверить таблицы.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

Схема полностью воспроизводится из репозитория.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
