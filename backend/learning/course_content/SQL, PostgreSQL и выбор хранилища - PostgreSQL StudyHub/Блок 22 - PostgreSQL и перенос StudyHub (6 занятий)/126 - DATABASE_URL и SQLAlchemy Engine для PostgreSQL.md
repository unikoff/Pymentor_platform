## 126. DATABASE_URL и SQLAlchemy Engine для PostgreSQL

### Цель

Переключить database layer с SQLite на PostgreSQL через конфигурацию.

### Основное содержание

- формат PostgreSQL URL;
- driver;
- environment variable;
- Settings;
- sync Engine;
- pool как набор connections;
- `SELECT 1`;
- диагностика host/port/password;

### Практика StudyHub

Добавить driver, обновить `.env.example`, создать engine и проверить соединение.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

StudyHub создаёт соединение с PostgreSQL из Settings.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
