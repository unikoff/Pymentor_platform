# Этап 6. SQL, PostgreSQL и выбор хранилища

## Общая цель

Этап снимает магию ORM: ученик видит SQL под SQLAlchemy, переносит StudyHub на PostgreSQL, строит JOIN и агрегаты, выполняет транзакции, измеряет запросы и сравнивает модели хранения.

## Сквозной проект

`PostgreSQL StudyHub`

## Карта блоков

| Блок | Занятия | Название | Назначение |
| ---: | ---: | --- | --- |
| 21 | 117–122 | SQL как язык работы с данными | Связать SQLAlchemy с реляционной моделью и ручным SQL. |
| 22 | 123–128 | PostgreSQL и перенос StudyHub | Перенести рабочий API с SQLite на PostgreSQL без изменения HTTP-контракта. |
| 23 | 129–134 | JOIN, агрегаты и транзакции | Перейти от простого CRUD к связанным данным, статистике и атомарным операциям. |
| 24 | 135–140 | Индексы, планы запросов и модели хранения | Научиться измерять запросы и выбирать хранилище по требованиям. |

## Методика этапа

```text
проблема → минимальная модель → ручной эксперимент → применение в StudyHub
→ ошибочный сценарий → тест/измерение → объяснение результата
```
