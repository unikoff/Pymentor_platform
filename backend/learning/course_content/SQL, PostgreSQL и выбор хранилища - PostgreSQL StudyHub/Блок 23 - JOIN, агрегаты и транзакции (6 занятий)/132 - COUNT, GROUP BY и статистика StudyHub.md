## 132. COUNT, GROUP BY и статистика StudyHub

### Цель

Получать показатели вместо списка объектов.

### Основное содержание

- `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`;
- `GROUP BY`;
- неагрегированные колонки;
- COUNT(*) vs COUNT(column);
- group после JOIN;
- нулевые значения;
- label результата;
- SQLAlchemy func;

### Практика StudyHub

Посчитать все и завершённые задачи по пользователю и средний приоритет по категории.

### Визуальная подача

Использовать схему движения данных, карточки ролей, сравнение корректного и ошибочного подхода, пошаговый разбор и одну доминирующую интерактивную механику.

### Результат занятия

Появляется первый статистический endpoint.

### Критерии готовности

- ученик объясняет главную модель своими словами;
- запускает минимальный пример;
- изменяет один параметр и предсказывает эффект;
- проверяет успешный и ошибочный сценарий;
- фиксирует результат отдельным Git-коммитом;
