# Pymentor Platform

**Pymentor Platform** — образовательная платформа для самостоятельного изучения Python и backend-разработки. Здесь собрана последовательная программа: теория, видео, практика, проверяемые задания и личный прогресс в одном интерфейсе.

Это открытый исходный код платформы. Репозиторий показывает, как устроены учебные треки, доступ к материалам, практика и административная часть.

## Для кого

- Для начинающих, которые хотят пройти путь от основ Python до FastAPI, баз данных, авторизации, Docker и подготовки к собеседованиям.
- Для учеников Pymentor, которым нужен единый кабинет с программой, практикой и записями на занятия.
- Для преподавателей и администраторов, которые ведут учебные материалы, доступы и расписание.
- Для разработчиков, которым интересен пример компактной образовательной платформы на FastAPI и React.

## Что умеет платформа

- Учебные курсы с блоками и уроками: от основ Python до backend-разработки и деплоя.
- Теория в Markdown, индивидуальные YouTube-видео для уроков и переход к следующему занятию.
- Практика с ручными заданиями и задачами с автоматической проверкой кода.
- Отметки о прохождении и общий прогресс по курсу.
- Регистрация, вход по сессии и управление доступом к урокам по подписке.
- Карта обучения, открытая для навигации по каждому курсу.
- Календарь записи на занятия с преподавателем и месячные лимиты.
- Административная панель: аккаунты, подписки, лимиты и расписание.

## Технологии

- Backend: Python, FastAPI, SQLAlchemy, Pydantic, SQLite.
- Frontend: React, TypeScript, Vite.
- Сессии и доступ: HTTP-only cookie, роли администратора и подписка.
- Production: собранный frontend раздаётся FastAPI как SPA.

## Структура репозитория

```text
backend/                         FastAPI, база данных, API и учебная логика
backend/learning/course_content/ Учебные треки и Markdown-материалы
frontend/                        React-приложение
frontend/dist/                   Production-сборка frontend
.github/workflows/               Проверка и выпуск production-сборки
```

## Быстрый запуск

Нужны Python 3.12+ и Node.js 20+.

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\python.exe -m pip install -r requirements.txt

cd frontend
npm install
cd ..
.\start.bat
```

После запуска:

- frontend: `http://127.0.0.1:5173`
- API: `http://127.0.0.1:8001`

Для Linux используйте `./start.sh`.

## Production-сборка frontend

Если менялся frontend, перед выпуском соберите его с production-префиксом:

```powershell
cd frontend
$env:VITE_BASE = "/platform/"
npm run build
```

Каталог `frontend/dist` намеренно хранится в репозитории: production-хостинг получает уже готовую сборку. Детали конкретной production-инфраструктуры и деплоя не публикуются в этом README.

## Учебный контент

Материалы лежат в `backend/learning/course_content/`. У каждого урока может быть отдельное видео: в конце Markdown-файла добавьте строку:

```md
<!-- youtube: https://youtu.be/VIDEO_ID -->
```

Если ссылка не указана, блок видео не отображается.
